# pré_requis: Nécessite python 3.10 au moins ainsi que l'accès à la librairie standard de python.Le programme dépend juste de `secrets` et pour être plus précis de sa fonction choice. 

# Utilisation
pour l'utiliser on peut :
## -**par import**
```python
from main import main, game_setup
```
## -**par ligne de commande**
`python main.py`
